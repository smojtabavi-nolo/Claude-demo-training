# Labs

**Self-paced labs for the Claude Code one-day intensive.** Setup is required pre-work — complete it via the [pre-work checklist](../docs/ONE-DAY-PREWORK-CHECKLIST.md); **Lab 00 walks through that same setup as a hands-on lab.** The module labs (01–06) are optional practice for each module that you can work through at your own pace, or run ahead with. They take longer than the live session allows, so the module labs are not required to keep up with the day.

## Enrichment labs

| # | Lab | Module |
| --- | --- | --- |
| 00 | Setup & local stack bootstrap *(required setup)* | — |
| 01 | Know the machine: Claude Code primitives | 1 |
| 02 | From mess to mission: creating plans | 2 |
| 03 | Commit archaeology: find the why | 3 |
| 04 | Give Claude eyes & ears: dynamic data | 3 |
| 05 | Review, test & verify | 4 |
| 06 | Agent orchestra: multi-agent workflow | 5 |

Do the module labs in order when you have time — each builds artifacts the next one reuses. The module labs (01–06) are not required to keep up in the live session; treat them as practice you can run ahead with or revisit afterward.

## Optional further practice

- **07 — Build Your Agentic Operating System:** optional further practice that makes the Lab 06 workflow observable and self-improving. Not part of the one-day course.
